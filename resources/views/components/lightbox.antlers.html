{{# Full-screen image lightbox. Driven by Alpine `lb` state that must live on an ancestor: x-data="{ lb: { open: false, src: '', alt: '' } }". Triggers set lb.src / lb.alt and flip lb.open. The overlay dims the page behind it; clicking the backdrop, pressing Escape or the close button closes it. The image scales to the viewport (contain, minus a small margin so the corner button stays on screen). The close button is centred exactly on the image's top-right corner — the corner (and thus the image diagonal) runs through its centre. It reuses the menu toggle style (menu/buttons/hide). #}}
